# ✨ Fundo Mágico

Uma aplicação web desenvolvida com HTML, CSS e JavaScript que utiliza Inteligência Artificial integrada via n8n para gerar backgrounds animados a partir de descrições em texto.

O usuário descreve o efeito visual desejado e a aplicação retorna o código HTML e CSS correspondente, além de exibir uma pré-visualização em tempo real.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da interface da aplicação.
- **CSS3:** Estilização da página, responsividade e criação da experiência visual.
- **JavaScript:** Implementação da lógica da aplicação, manipulação do DOM e tratamento de eventos.
- **n8n:** Automação do fluxo responsável por receber a descrição do usuário, processar a solicitação e retornar o código gerado pela IA.

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento Front-End, incluindo manipulação do DOM, consumo de APIs, requisições assíncronas com Fetch API e integração com workflows de IA utilizando n8n.

O projeto foi construído durante a **Semana do Zero ao Programador Contratado**, promovida pelo canal **Dev em Dobro**, como forma de aplicar na prática conhecimentos de desenvolvimento web moderno.
